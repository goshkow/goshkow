## Чем я занимаюсь

Мне нравится делать продукты, где не надо долго разбираться, как вообще это запустить.
Чтобы было:
- понятно
- быстро
- удобно
- и без ощущения, что ты воюешь с конфигами вместо того, чтобы решать задачу

Сейчас мой фокус в основном такой:
- кроссплатформенные приложения
- утилиты с нормальным UI
- AI/LLM-инструменты, в том числе локальные
- серверная часть и автоматизация
- open-source софт, который реально хочется использовать
